BATMON
Batmon is a laptop battery monitoring application

Installation:
Run the batmon_setup.exe file to install the application

Usage:

Run the batmon.exe file

Batmon will alert the user if

1.Battery percent is below 10% and not charging(repeated after every 1 mins)
2.Battery percent is below 20%
3.Battery percent has reached 90%
4.Battery percent has reached 100% and still charging(repeated alert every 1 min)
