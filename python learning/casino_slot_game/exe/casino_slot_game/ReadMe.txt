$$$ CASINO SLOT GAME $$$
CASINO SLOT GAME is a arcade game simulation of slot machine game

Installation:
Run the csg_setup.exe file to install the application

Usage:

Run the csg.exe file

1. Insert coin using the bottom left red button
2. Spin the slots using bottom right red button
3. Cash checkout button can be selected to quit at any time
4. Audio can be switched on and off using the sound button
5. Once coins are over, use the green top-up button to restart the game
